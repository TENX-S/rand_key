## Try it!
```shell script
# Default case: amount of letters: 10, symbols: 2, numbers: 3
$ cargo run --release --example grp_test

$ cargo run --release --example grp_test 16 2 3

# Try a larger number!
$ cargo run --release --example grp_test 200000 200 300
```
